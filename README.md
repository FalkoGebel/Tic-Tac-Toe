# Tic Tac Toe
A simple Tic Tac Toe board for two human players to play. There is no computer player and no logic for winning or losing.

## Programming Language
- C#

## UI
- WPF

## Features
- Selection of symbol and color for each player
- Occupy squares
- Reset board
